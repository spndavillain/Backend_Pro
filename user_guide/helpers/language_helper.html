Extended Language Helper

The Extended Language Helper contains functions that assist in working with language lines.

Note: This file is initialized automatically by the system so there is no need to do it manually.

Function Reference

translate_lang('line')

The function will translate a string into its correct value based on the value stored in your language file OR just return the string. Lets look at an example:

// Lets say we have the following language line
$lang['preference_field'] = 'Preference Field';

// By doing the following we can output the language line
print translate_lang('lang:preference_field');

// But the following would return the same value
print translate_lang('Preference Field');

Now your probably wondering why this is useful, why can't we just use lang() instead? The difference is lang() will retrun FALSE if it can't find a matching language line. Where as this would return what you just gave it.

In BackendPro this is used to allow the user to specify either the string directly to use or the lang line to load (with minimum extra code).
