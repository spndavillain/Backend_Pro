Extended Date Helper

The Extended Date Helper contains functions to help with translating MySQL DateTimes to other formats.

Function Reference

mysqldatetime_to_unix('datetime')

Converts a MySQL datetime in the format (YYYY-MM-DD hh:mm:ss or YYYY-MM-DD) to a Unix timestamp.

$datetime = '2010-10-03 12:08:22';
$unix_timestamp = mysqldatetime_to_unix($datetime); // Result is 1286107702

mysqldatetime_to_date('datetime', 'format')

Converts a MySQL datetime in the format (YYYY-MM-DD hh:mm:ss or YYYY-MM-DD) to a date format of your choice.

$datetime = '2010-10-03 12:08:22';
$unix_timestamp = mysqldatetime_to_date($datetime, 'd/m/Y'); // Result is 3/10/2010
