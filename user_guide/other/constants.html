Extended Constants

The extended constants file provides three new constants for use throughout your code. It extendeds the default constant file found in application/config.

Note: This file is initialized automatically by the system so there is no need to do it manually.

Usage

BASE_URL

This will return the base URL of your website. For example

// If your site is located at the root of http://localhost it will return
http://localhost/

// If your site is located in a sub-directory of http://localhost/BackendPro/ it will return
http://localhost/BackendPro/

BASE_URI

This will return the base URI of your website. For example

// If your site is located at the root of http://localhost it will return
/

// If your site is located in a sub-directory of http://localhost/BackendPro/ it will return
/BackendPro/

APPPATH_URI

This will return the application path URI of your website. For example

// If your application folder is located at http://localhost/system/application/ it will return
/system/application/

// If your application folder is located in a sub-directory of http://localhost/BackendPro/system/application it will return
/BackendPro/system/application/
