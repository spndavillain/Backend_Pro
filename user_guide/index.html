Welcome to BackendPro

BackendPro is a Control Panel toolkit designed to make creating your sites quick and easy. It has been designed for advanced CodeIgniter developers in mind, but with the help of the user guide and articles there is no reason a CodeIgniter begginer can't also use it. Built ontop of the excellent CodeIgniter Framework it allows you to focus on the front-end part of your site. No worries about authentication or how to manage the site, this is done for you out of the box.

Who is BackendPro For?

BackendPro is right for you if:

  • You want to use the CodeIgniter Framework for your site.
  • You don't want to worry about how to allow users to login.
  • You want powerful yet simple member group access control.
  • You require css/js asset controls.
  • You want powerful features straight out of the box.
  • You just want to start making your site, not worrying about the code which runs it.
  • You need clear, thorough documentation.
