Installation Instructions

BackendPro comes with an installer to setup your system. Before we can run it we need to do a few manual steps.

  1. Download the latest version of CodeIgniter and unzip it to a folder on your server.
  2. Download the latest copy of BackendPro, details can be found on the Downloading BackendPro page.
  3. Unzip the contents the zip file into the same directory containing your CodeIgniter files. Some files may be overwritten.
  4. Navigate to the install directory in your browser and follow the onscreen instructions.

You should now have a fully working version of BackendPro running on the CodeIgniter framework. If you don't and you had an issue please first try the troubleshooting page, and if you still have no luck please contact me on the BackendPro forum thread and we can hopefully fix the problem and get you up and running.
