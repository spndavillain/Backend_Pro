Subversion Server

Subversion is a version control system.

Public subversion access is now available via http://svn2.assembla.com/svn/backendpro/trunk/ please note that while every effort is made to keep this codebase functional, we cannot guarantee the functionality of code taken from the repository.

Beginning with version 0.2, version snapshots (tags) are also available via the Subversion server via http://svn2.assembla.com/svn/backendpro/tags/.
