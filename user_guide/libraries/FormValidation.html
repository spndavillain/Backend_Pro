Extended Form Validation Class

The Form Validation class extends the base CodeIgniter Form Generation Class. It provides aditional methods so the Form Validation class can utilise the BackendPro Status Class.

Note: This class is initialized automatically by the system when you load the Form Validation Class.

Usage Examples

Below is a full example of how you would use this extended class.

$this->load->library('form_validation');

$this->form_validation->set_rules(....);

if($this->form_validation->run() === TRUE)
{
        // Save form to Database
}
else
{
        // Load errors into status messages
        $this->form_validation->transfer_errors();
        
        $this->load->view('input_form');
}

Function Reference

$this->form_validation->transfer_errors()

The function transfers all validation errors from the Form Validation Class to the Status Class so they can be displayed along with other error messages.

$this->form_validation->transfer_errors();
