Setting Library

The setting library allows you get and set setting values from/to the database. Settings are very like values stored in the CodeIgniter Config. The difference is as settings are stored in the database it means they can be changed by a user easily via the control panel. For more details on managing your settings please Setting Management.

Note: This class is initialized automatically by the system so there is no need to do it manually.

Function Reference

$this->setting->item()

Fetch a setting value from the database.

$this->setting->item('slug')

By default the setting will be fetched from the cache, if you want to load it again from the database you can do this by using the second optional parameter.

$this->setting->item('slug', FALSE);

$this->setting->set_item()

Set a new value for a setting.

$this->setting->set_item('slug', 'value')
