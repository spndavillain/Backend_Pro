BackendPro Class

The BackendPro Class is the main super class for the BackendPro system. It sets up the system and loads all needed librarys/helpers. This class should be included in the autoload config file.

Important:  This class is initialized automatically by the system so there is no need to do it manually.

How to use the Backendpro Class?

BackendPro Class is loaded by default by the BackendPro Class. The class file can be found at system/application/libraries/BackendPro.php

Configuration Settings

All configuration settings for the BackendPro Class can be found in system/config/backendpro.php.

Preference Default Value Options Description
backendpro_table_prefix be_ None Default table prefix used by BackendPro database tables
backendpro_template_dir NULL None View file base directory
backendpro_template_public public/ None Public view file directory
backendpro_template_admin admin/ None Admin view file directory
