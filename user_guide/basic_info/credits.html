Credits

BackendPro was created and designed by Adam Price due to the need of a simple control panel which could be the base of any new website.

The majority of code included in the application was written by Adam Price but there are ideas and some snippets of code written by other parties. This page is to give credit where it is due.

FamFamFam Icons

Throughout BackendPro a wide range of their icons have been used to bring colour to boring operations and aid the user into what an action does. http://www.famfamfam.com/

Phil Sturgeon

The BackendPro installer, MY_Model and constants file is based on some code found in PyroCMS written by Phil Sturgeon.

BackendPro Shield Icon

The icon used for BackendPro was created by http://pixel-mixer.com.

CSS Reset Stylesheet

Based on work done by Eric A. Meyer.

Extended Date Helper

The date helper has been extending by using some of the functions found in Clemens Kofler's Date Helper.
